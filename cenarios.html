📐 Lógica de Construção dos Cenários

  • Pessimista: Clima adverso (La Niña forte) + queda de preços + mantém todas as operações deficitárias + inflação de insumos
  • Base: Clima normal + preços estáveis + reduz Cana SP em 50% + elimina parcialmente DM 82I78
  • Otimista: Clima favorável + câmbio favorável + elimina Cana SP + elimina DM 82I78 + expande variedades top + reduz custos ADM
  • Produtividade base (65 sc/ha): Média das últimas 5 safras = 63,04 sc/ha → arredondado para cima pela tendência positiva
  • Preços base (R$ 115/sc soja): Média CBOT Mar/26 ~US$10,50/bu + prêmio + câmbio R$ 5,80

🌪️ Insights da Análise de Sensibilidade

  • Variável #1 (maior impacto): Preço da soja — cada R$ 10/sc = ~R$ 1,8M no resultado (a 65 sc/ha)
  • Variável #2: Decisão sobre Cana SP — impacto binário de R$ 2,31M
  • Variável #3: Produtividade da soja — cada 5 sc/ha = ~R$ 1,6M (a R$ 115/sc)
  • Variável #4: Custo de insumos — cada 10% = ~R$ 1,0M no resultado total
  • Break-even soja (cenário base): 58,1 sc/ha a R$ 115/sc → folga de 7 sc vs. projeção base
  • Zona de conforto: Soja ≥ 65 sc/ha + PMV ≥ R$ 115/sc + Sem Cana SP = resultado ≥ R$ 6M
